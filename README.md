# AI Code Reviewer

An AI-powered code reviewer for Java and Node.js code built 
using Node.js, Express, and local LLM via Ollama.

## Features
- Security vulnerability detection
- Performance issue identification  
- Best practice recommendations
- Code quality scoring (1-10)
- Suggested fixes for critical issues

## Tech Stack
- Node.js + Express (backend)
- Ollama + llama3.2 (local AI)
- HTML/CSS/JavaScript (frontend)

## Setup
1. Install Ollama from ollama.com
2. Run: ollama pull llama3.2
3. Clone this repo
4. cd backend && npm install
5. node server.js
6. Open http://localhost:3000
